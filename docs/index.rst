GOMet documentation!
==================================

GOMet (\\ɡɔ.mɛt\\) is a python module gathering some global optimization methods 
commonly used in geophysical inversion: Simulated Annealing (SA), Genetic
Algorithm (GA), Particle Swarm Optimization (PSO) and Neighbourhood
Algorithm (NA).
